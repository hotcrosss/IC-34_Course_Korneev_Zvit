Пояснення

Для мобільних телефонів (@media (max-width: 480px)) Цей блок стилів застосовується для екранів, ширина яких менша або дорівнює 480px, що зазвичай відповідає смартфонам. Розмір тексту зменшується, щоб він помістився на невеликому екрані (заголовок h1 → 25px, кнопки → 12px). Відступи та розміри кнопок зменшуються, щоб забезпечити зручність користування. Гнучке вирівнювання: елементи, такі як заголовки та навігація, розташовуються вертикально (flex-direction: column) для більш компактного вигляду. Зображення у футері автоматично змінює розмір до 200px, щоб не виходити за межі екрану. Менша відстань між пунктами меню футера (gap: 3px).
