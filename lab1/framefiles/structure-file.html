Структура документа

1. Документні метадані

  • Тип документа: <!DOCTYPE html>
  • Тег <html lang="en">
  • Тег <head>:
    • <meta charset="UTF-8"> для кодування символів
    • <meta name="viewport" content="width=device-width, initial-scale=1.0"> для масштабування сторінки
    • Тег <title> з назвою "Document"

2. Тіло документа

  • Тег <body> містить основні елементи сторінки:

  • Заголовок (header)

    • Тег <header>
    • Тег <nav> для навігаційного меню:
      • <ul> — список
      • <li> — пункт меню: "Про нас", "Курси", "Контакти"

  • Основний контент (main)

    • Тег <main> містить:
      • <h1> — "Лабораторна робота 1"
      • <h2> — "Корнєєв Михайло"
      • <img src="img/images.jfif" alt="#"> — зображення
      • Таблиця <table> з інформацією про курси:
        • <th> — заголовки стовпців: "Назва курсу", "Термін", "Ціна"
        • <td> — дані курсів: HTML, CSS, JavaScript тощо
      • <h2> — "Roadmap CSS"
      • Невпорядкований список <ul>:
        • Learn the basics
        • Making Layouts
        • Responsive design
      • <h2> — "Roadmap JavaScript"
      • Упорядкований список <ol>:
        • Learn the basics
        • Learn DOM Manipulation
        • Fetch API
      • Форма <form> з полями введення:
        • Поле тексту: <input type="text" name="firstname"> для імені
        • Поле тексту: <input type="text" name="lastname"> для прізвища
        • Кнопка: <input type="submit"> для надсилання форми
      • <h1> — "Посилання"
      • Посилання: <a href="zvit.html"> на звіт

  • Футер (footer)

    • Тег <footer> з невпорядкованим списком:
      • Інстаграм
      • Телеграм
