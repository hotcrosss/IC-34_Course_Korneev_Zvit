Висновок

Під час виконання лабораторної роботи було набуто практичні навички роботи з HTML-документами. Було вивчено базову структуру HTML-документу та застосовано ключові елементи, такі як заголовки, списки, таблиці, зображення, посилання та форми. Створено шаблон звітного документа, який може використовуватись для відображення результатів роботи з усіх лабораторних завдань. Цей шаблон містить різноманітні елементи для організації інформації, включаючи списки з різними значками на різних рівнях вкладеності, таблиці для структурованого представлення даних, форми для введення користувачем інформації та навігаційні елементи для зручного переміщення по документу. Навички, отримані під час роботи з HTML, є необхідними для подальшого використання у веб-розробці та створенні структурованих веб-сторінок. Таким чином, мета лабораторної роботи досягнута, і створено звітний шаблон, який можна використовувати в усіх наступних лабораторних роботах.
